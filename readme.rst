#############
O que é o CIP
#############

O CIP é o Centro Integrado de Pesquisa, que tem como objetivo concentar as informações sobre pesquisa dentro da Universidade.


*******************
Server Requirements
*******************

PHP versão 5.4 ou mais nova é recomendada, o SGBD utilizado é o MySQL.

It should work on 5.2.4 as well, but we strongly advise you NOT to run
such old versions of PHP, because of potential security and performance
issues, as well as missing features.
